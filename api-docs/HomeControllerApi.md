# HomeController API 명세서

## 개요
이 컨트롤러는 애플리케이션의 메인 페이지(홈) 요청을 처리하며, 사용자 로그인 상태에 따라 다른 뷰를 제공합니다.

## API 목록

### 1. 메인 홈 페이지 조회

*   **설명:** 애플리케이션의 메인 홈 페이지를 조회합니다. 로그인 상태에 따라 다른 페이지를 반환합니다.
*   **URI:** `/`
*   **HTTP Method:** `GET`
*   **인증:** 선택적 (로그인 여부에 따라 응답이 달라짐)

#### 요청

*   **Header:**
    *   `Cookie`: `SESSION` (로그인된 경우 세션 ID 포함)

#### 응답

*   **성공 (HTTP 200 OK):**
    *   **로그인되지 않은 경우:** `home.html` 뷰 페이지 반환
    *   **로그인된 경우:** `loginHome.html` 뷰 페이지 반환 (뷰에 `loginMember` 객체 전달)
